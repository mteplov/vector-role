# Vector Role

Роль устанавливает и настраивает Vector 0.55.0 на Linux (Ubuntu/Debian).

## Описание

Vector — это высокопроизводительный инструмент для сбора, обработки и передачи логов.

Данная роль:
- устанавливает Vector из официального tar.gz архива
- создаёт systemd сервис
- разворачивает конфигурацию
- запускает сервис

---

## Требования

- Ubuntu / Debian
- доступ в интернет для загрузки пакета
- systemd

---

## Переменные

Роль не использует обязательные переменные.

Опционально можно расширить:

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| vector_version | версия Vector | 0.55.0 |
| vector_config_path | путь к конфигу | /etc/vector |

---

## Структура роли

- tasks/ — основные задачи установки
- templates/ — конфигурация vector.toml
- handlers/ — перезапуск сервиса
- defaults/ — переменные по умолчанию
- vars/ — статические переменные
